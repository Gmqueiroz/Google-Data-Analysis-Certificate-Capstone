## 📈 Google Data Analytics Capstone Project

Este projeto foi desenvolvido como estudo de caso final do certificado profissional de análise de dados da Google, utilizando dados reais de compartilhamento de bicicletas da Cyclistic.

O objetivo da análise foi identificar diferenças comportamentais entre usuários casuais e assinantes, buscando entender quais fatores poderiam incentivar clientes ocasionais a se tornarem membros recorrentes.

Esse projeto foi desenvolvido há mais de 3 anos e marcou meu primeiro estudo de caso completo na área de análise de dados. Na época, optei por realizar praticamente toda a preparação, análise e visualização diretamente no Power BI.

Hoje, olhando retrospectivamente, vejo que essa não foi a melhor decisão para sustentar alguns dos insights apresentados. Parte da modelagem, tratamento e exploração poderia ter sido feita de maneira mais robusta utilizando SQL ou Python antes da camada visual. Além disso, algumas escolhas de visualização e organização dos dashboards poderiam ter sido estruturadas de forma mais clara e analiticamente consistente.

Ainda assim, o projeto teve um papel importante no desenvolvimento das minhas bases em:
- limpeza e preparação de dados;
- análise exploratória;
- construção de dashboards;
- comunicação de insights de negócio;
- pensamento analítico orientado por dados.
